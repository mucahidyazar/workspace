## üöÄ Installation

<hr />

1. Install [Node.js](https://nodejs.org/en/)
2. Add all environment variables (Which env for what I described them in
   .env.example)
3. After you add your env variables then run the codes on terminal

```bash
pnpm install && pnpm dev
```

4. Go localhost:3000
